# Pixlato

> **Note**: 이 프로젝트는 비전문 개발자의 개인 취미 프로젝트이며, **Google Antigravity**를 주력으로 다수의 AI 모델을 활용하여 개발 및 관리되고 있습니다(AI-Driven). 현재 개발 중인 단계로 기능의 정상적인 동작을 보장하지 않으며, 사용 시 예기치 않은 오류가 발생할 수 있습니다. 본 문서 또한 AI 에이전트에 의해 작성되었습니다.

이미지를 픽셀 아트로 변환하고 관리하는 도구입니다. 다수의 이미지를 목록 형태로 불러와 동일한 스타일로 한 번에 처리할 수 있는 기능을 제공합니다.

## 안내 사항

- **제공 형태**: 현재 이 프로그램은 패키징된 실행 파일(.exe 등) 형태로 제공되지 않습니다. 소스 코드를 직접 실행해야 하며, 향후 패키징된 배포판을 제공할 예정입니다.

## 주요 기능 (무엇을 할 수 있나요?)

- **이미지 픽셀 아트 변환**: 일반 사진이나 그림을 사용자가 지정한 해상도와 색상 제한에 맞춰 정교한 픽셀 그래픽으로 바꿉니다.
- **다중 파일 목록 관리 및 일괄 처리**: 여러 장의 이미지를 목록에 올려두고 편리하게 전환하며 작업할 수 있습니다. 현재 설정된 변환 스타일을 목록 내 모든 이미지에 동일하게 적용하여 한 번에 결과물을 뽑아낼 수 있습니다.
- **애니메이션(GIF) 프레임 분해 및 재조합**: 움직이는 GIF 파일을 개별 프레임으로 분해하여 일괄적으로 픽셀화한 뒤, 다시 하나의 애니메이션 파일로 합칠 수 있습니다.
- **투명 배경 및 외곽선 처리**: 이미지 모서리 색상을 감지해 배경을 투명하게 만들거나, 캐릭터 주위에 뚜렷한 외곽선을 추가하여 가독성을 높입니다.
- **이미지 세부 디테일 강화**: 픽셀화 과정에서 뭉개지기 쉬운 눈, 코, 입 등 주요 특징들을 변환 전 미리 강화하여 디테일을 살립니다.
- **팔레트 설계 및 추출**: 이미지에서 가장 잘 어울리는 색상을 자동으로 뽑아내거나, 컬러 피커를 통해 나만의 색상 조합을 직접 설계하여 이미지에 입힐 수 있습니다.

## 기술 스택

- **언어**: Python 3.13
- **GUI**: CustomTkinter (현대적인 UI 환경)
- **이미지 처리**: Pillow
- **수치 연산 및 가속**: PyTorch, NumPy (하드웨어 가속 지원)

## 설치 및 실행

### 요구 사항
- Python 3.13 이상

### 설치
1. 저장소를 클론합니다.
   ```bash
   git clone https://github.com/your-repo/Pixlato.git
   cd Pixlato
   ```
2. 필요한 패키지를 설치합니다.
   ```bash
   pip install -r pixel_crafter_gui/requirements.txt
   ```

### 실행
```bash
python pixel_crafter_gui/main.py
```

## 구조 및 특징

- **로직 분리**: 이미지 처리 로직(`core`)과 사용자 인터페이스(`ui`)가 분리되어 있어 구조가 명확합니다.
- **일괄 처리 최적화**: 여러 자산에 동일한 픽셀 아트 스타일을 빠르게 적용하는 배치 프로세싱에 최적화되어 있습니다.
- **실시간 확인**: 설정 값을 변경하는 즉시 변환된 결과를 프리뷰 화면을 통해 확인할 수 있습니다.
