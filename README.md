# 🎨 Pixlato

**Pixlato(픽슬라토)**는 이미지를 픽셀 아트 스타일로 변환하는 도구입니다. 여러 알고리즘과 AI 기술을 활용하여 픽셀 에셋 제작 과정을 돕습니다.

---

## 🤖 AI 주도 개발 (AI-Driven Development)
본 프로젝트는 **AI 에이전트와 사용자의 협업**을 통해 설계 및 구현되고 있습니다. 프로그램의 구조 설계, 알고리즘 작성, 인터페이스 정제 등 개발의 전 과정에 AI가 참여하고 있는 실험적인 프로젝트입니다.

---

## ✨ 주요 기능

### 1. 색상 변환 엔진
*   **기하학적 색역 보존**: 이미지의 색상 분포를 분석하여 하이라이트나 주요 원색이 누락되지 않도록 돕습니다.
*   **지각적 LAB 매핑**: CIE LAB 색상 공간을 사용하여 시각적으로 유사한 색상을 연결합니다.
*   **자동 최적화**: 이미지 특성에 맞춰 색상 수와 노이즈 필터 강도를 제안하는 기능을 포함합니다.
*   **공간 정제**: 전처리 및 후처리 필터를 통해 영역 내의 불필요한 도트 노이즈를 줄여줍니다.

### 2. 배치 프로세싱 및 인벤토리
*   **인벤토리 시스템**: 여러 이미지를 불러와 관리하며 개별 또는 일괄 설정을 적용할 수 있습니다.
*   **규격화 (Normalization)**: 해상도가 다른 이미지들을 하나의 크기로 통일하여 내보낼 수 있습니다.
*   **GIF 병합**: 인벤토리의 이미지들을 하나의 애니메이션 GIF 파일로 합쳐서 저장합니다.

### 3. 시각 보정 기능
*   **배경 제거**: AI 기반 자동 제거 기능과 사용자가 직접 지점을 지정하는 마킹 도구를 제공합니다.
*   **알파 채널 정제**: 배경 제거 후의 경계선을 정리하여 외곽선이 깔끔하게 그려지도록 돕습니다.

### 4. 기타 워크플로우 지원
*   **다양한 포맷 지원**: PNG, TGA, WebP, TIFF 등 게임 제작 환경에서 쓰이는 형식을 지원합니다.
*   **메타데이터 활용**: 작업 설정을 PNG 파일 내부에 포함하여 나중에 다시 불러올 수 있습니다.
*   **플러그인 시스템**: 별도의 스크립트를 통해 기능을 추가할 수 있는 구조를 갖추고 있습니다.

## 🚀 시작하기 (Quick Start)

> **⚠️ 고지 사항**: 본 프로그램은 현재 설치형 실행 파일(.exe 등)을 제공하지 않습니다. 프로그램을 실행하려면 아래의 안내에 따라 **Python 개발 환경**이 구축되어 있어야 합니다. (권장 사양: Python 3.13 이상)

### 1. 가상환경 설정
리포지토리를 클론한 후, 프로젝트 루트 디렉토리에서 아래 명령을 실행합니다.

```bash
# 리포지토리 클론
git clone https://github.com/your-repo/pixlato.git
cd pixlato

# 가상환경 생성
python -m venv .venv

# 가상환경 활성화 (Windows)
# PowerShell에서 차단될 경우: Set-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope Process
.venv\Scripts\activate

# 가상환경 활성화 (macOS/Linux)
# source .venv/bin/activate
```

### 2. 의존성 설치
필요한 라이브러리를 설치합니다. (용량이 큰 AI 모델 관련 패키지가 포함되어 있습니다.)

```bash
python -m pip install --upgrade pip
pip install -r pixel_crafter_gui/requirements.txt
```

### 3. 프로그램 실행
메인 스크립트를 실행하여 프로그램을 구동합니다.

```bash
python pixel_crafter_gui/main.py
```
